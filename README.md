# EXPENSE-TRACKER
A project made by me as a 1st year student which actually helps people around the world especially college student like me.
 # Features of this project

- 🔐 User Authentication (Register & Login)
- ➕ Add, Edit, and Delete Expenses
- 💰 Daily Budget Limit Tracking
- 📊 Budget vs Spending Analysis
- ⚠️ Alerts when Budget Limit is Exceeded
- 📈 Analytics Dashboard for Expense Insights
- 🗂️ Category-wise Expense Tracking
- 📅 Date-wise Expense Management
- 🔁 Support for Recurring Expenses
- 🌍 Multi-Currency Support
- 🔥 Real-time Data Storage using Firebase
- 💻 Responsive and User-Friendly Interface
